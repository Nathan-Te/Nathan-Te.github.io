A propos

ZikiFy a été créé par trois étudiants en 3ème année de Licence Informatique
à l'Université du Mans dans le cadre d'un projet sur le module Architecture logicielles
pour le développement d'applications Web.

ZikiFy permet à l'utilisateur :
- d'écouter de la musique via YouTube ou Spotify,
- de trouver les paroles d'une chanson, - de trouver des infos sur un artiste, - de trouver des dates de concerts.

L'application fonctionne grâce à plusieurs API : Spotify, YouTube, Lyrics.ovh, music-story et Ticketmaster.
